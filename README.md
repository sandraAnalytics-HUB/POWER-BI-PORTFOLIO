PROJECT OVERVIEW:
This project analyzes historical aviation crash data to uncover trends, patterns, and safety insights across aircraft operators, locations, routes, and aircraft types. The dashboard is build using Power BI to transform raw crash records into actionable insights through interactive visualizations and key performance indicators (KPIs). The goal of this projects is to provide a comprehensive overview of aviation accidents and fatalities while demonstrating data cleaning, modelling, DAX calculations, and data visualization skills. 
OBJECTIVES:
The dashboard answers key questions such as: 
• How have aviation crashes changed over time? 
• Which years recorded the highest number of crashes and fatalities. 
• Which locations experience the most aviation accidents. 
• Which operators are involved in the highest crash frequencies? 
• What is the overall mortality rate across recorded crashes? 
• Which month and days of the week experience the most accidents. 
DATA INFORMATION:
The dataset contains historical aviation crash records with the following fields:

Columns Description Index Unique record identifier,Date Date of crash, Time Time of incident, Location Crash Location, Operator Airline or Aircraft operator, Flight Flight number, Route Flight Route, Type Aircraft type, CN / IN Aircraft identification number, Abroad Total passengers, Fatalities Total Fatalities, Ground Ground Fatalities.

KEY PERFORMANCE INDICATORS (KPIs):
The dashboard includes: 
• Total crashes 
• Total fatalities 
• Total passengers 
• Mortality rate 
• Average fatalities per crash 
• Total ground fatalities

KEY INSIGHTS 
• Aviation crashes have shown varying trends across different years. 
• Certain operators account for a larger share of recorded incidents. 
• Fatalities are concentrated in specific location and aircraft types. • The overall mortality rate indicates the severity of aviation accident within the dataset. 
• Seasonal pattern can be observed in monthly crash distribution.
